# Privacidad

El perfil, progreso, tarjetas, estadísticas, sesiones y grabaciones se guardan localmente. Si el usuario inicia sesión, los datos educativos —excepto grabaciones— se sincronizan a Firestore. No hay rastreadores, publicidad ni venta de datos.

El respaldo exporta solo datos educativos. No incluye contraseñas, tokens, configuración Firebase, datos financieros ni grabaciones. El usuario puede importar, reiniciar progreso o eliminar grabaciones desde Configuración.
