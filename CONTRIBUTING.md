# Contribución

Respete `AGENTS.md`. Cree cambios pequeños, agregue pruebas y ejecute typecheck, lint, tests, validación de contenido, build y guards. Cada lote pedagógico contiene tres semanas y debe pasar sus cuatro gates antes del siguiente.

El contenido debe ser original, estadounidense, apropiado al nivel y técnicamente correcto. No agregue cursos copiados, relleno ni enlaces que simulen funcionalidad.
